# goaustin
Interactive Austin Summer Guide featuring 101 curated experiences, immersive editorial design, responsive layouts and modern front-end web development.
🌆 Austin Summer 2026 – Interactive Travel Guide

An immersive, editorial-style web experience showcasing 101 curated ways to experience Austin and the Texas Hill Country during summer. Built with modern HTML, CSS, and JavaScript, the project combines stunning visuals, interactive discovery, and responsive design to create a premium travel-guide experience.
